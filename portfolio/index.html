1. Origami

Project Link: Origami Project

Tools Used: HTML5, CSS3, JavaScript, Bootstrap 5, jQuery, AJAX, AOS, Google Fonts, Google Sheets, FontAwesome

Details: Origami is an online platform designed to offer high-quality psychology courses at affordable prices. It focuses on providing comprehensive educational content, including crash courses for various competitive exams such as CUET, JMI, NET, and others.

Key Features:

  • User-friendly interface with modern design
  • Responsive layout compatible with various devices
  • Integration with Google Sheets for data management
  • Rich multimedia content enhanced by animations and interactive elements

Development: The website was built using a combination of HTML5, CSS3, JavaScript, and Bootstrap 5, alongside jQuery, AJAX, and AOS for dynamic content and animations. Google Fonts and FontAwesome were utilized for typography and icons, respectively.
